# ACCESS_KEY_READ

> Area: Access Key
> Roles: OWNER, ADMIN

Read the organization's access keys.

## Description

Grants read access to the organization's access keys — the credentials that authorize calls into
Pager's shared microservices and integrations. Held by OWNER and ADMIN.
