# 30002 AccessKeyNotBelongToOrganization

> Category: Access Key | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The access key exists but belongs to a different organization than the caller.

## Error Message

`Access key({}) does not belong to organization({})`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The access key id. |
| `{}` (2nd) | The caller's organization id. |

## When Returned

Thrown by [`AccessKeyRepository`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/repository/AccessKeyRepository.kt) when the [access key](/doc/tech_noun/TN0204_access_key.md) resolves but its [organization](/doc/tech_noun/TN0201_organization.md) is not the caller's — reached by `PUT /v1/access_key/{accessKeyId}` and `DELETE /v1/access_key/{accessKeyId}`.
