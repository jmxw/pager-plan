# 30003 AccessKeyUsedInProject

> Category: Access Key | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

Deletion was rejected because the access key is still used by projects.

## Error Message

`Access key({}) has been used in projects and cannot be deleted.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The access key id. |

## When Returned

Thrown by [`AccessKeyService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/AccessKeyService.kt) during `DELETE /v1/access_key/{accessKeyId}` when at least one [project](/doc/tech_noun/TN0301_project.md) still references the [access key](/doc/tech_noun/TN0204_access_key.md).
