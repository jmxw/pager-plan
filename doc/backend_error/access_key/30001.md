# 30001 AccessKeyIdNotExist

> Category: Access Key | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

No access key exists for the looked-up id.

## Error Message

`Access key id {} not exist.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The access key id that was looked up. |

## When Returned

Thrown by [`AccessKeyRepository`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/repository/AccessKeyRepository.kt) when an [access key](/doc/tech_noun/TN0204_access_key.md) is resolved by id and no matching row exists — reached by `PUT /v1/access_key/{accessKeyId}` and `DELETE /v1/access_key/{accessKeyId}`.
