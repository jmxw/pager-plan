# -9999 UnknownError

> Category: Unknown | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The catch-all error returned when a request fails with an internal or otherwise unhandled exception.

## Error Message

`Unknown error: {}`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The message of the underlying throwable. **Not substituted in practice:** [`CmsExceptionHandler`](/source/pager-backend/app/cms/src/main/kotlin/com/xwkj/cms/exception/CmsExceptionHandler.kt) constructs this error with no argument, so the response `errorMsg` reads the literal `Unknown error: {}`. |

## When Returned

Returned by [`CmsExceptionHandler`](/source/pager-backend/app/cms/src/main/kotlin/com/xwkj/cms/exception/CmsExceptionHandler.kt) as the fallback for (a) any `PagerInternalException` (after it is logged at the constant's log level) and (b) any exception not otherwise handled. The original error detail is written to the server log, not the response. The HTTP status is `500`.
