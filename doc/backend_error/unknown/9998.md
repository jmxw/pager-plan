# 9998

> Category: Unknown | HTTP Status: The Spring MVC status of the rejected request (a 4xx such as `400`, `405`, `406`, `415`); carried in `code`, with `errorCode` `9998`.

Returned when Spring MVC rejects a request around the controller with a 4xx status (malformed body, missing parameter, unsupported method or media type).

## Error Message

N/A — `9998` has no `PagerErrorMessage` constant and no fixed message. The response `errorMsg` is the message of the underlying Spring MVC exception.

## Placeholders

N/A — the message contains no placeholder.

## When Returned

Produced by [`CmsExceptionHandler.handleExceptionInternal`](/source/pager-backend/app/cms/src/main/kotlin/com/xwkj/cms/exception/CmsExceptionHandler.kt) for the exceptions Spring's `ResponseEntityExceptionHandler` handles whose resolved status is a 4xx; the response carries the hard-coded `errorCode` `9998` and the Spring status in `code`. 5xx cases on the same path are converted to [`-9999 UnknownError`](/doc/backend_error/unknown/-9999.md) instead. This code has **no enum constant** in `PagerErrorMessage` and no fixed message.
