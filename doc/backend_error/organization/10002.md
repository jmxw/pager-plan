# 10002 OrganizationIdentifierExist

> Category: Organization | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The organization identifier chosen at registration is already taken.

## Error Message

`Organization identifier {} exist.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The organization identifier submitted for registration. |

## When Returned

Thrown by [`OrganizationService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/OrganizationService.kt) during `POST /v1/organization/register` when the requested organization [identifier](/doc/tech_noun/TN0101_identifier.md) already belongs to an existing [organization](/doc/tech_noun/TN0201_organization.md).
