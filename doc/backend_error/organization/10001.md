# 10001 OrganizationIdNotExist

> Category: Organization | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

No organization exists for the looked-up id.

## Error Message

`Organization id {} not exist.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The organization id that was looked up. |

## When Returned

Thrown by [`OrganizationRepository`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/repository/OrganizationRepository.kt) when an [organization](/doc/tech_noun/TN0201_organization.md) is resolved by id and no matching row exists. The id is taken from the authenticated caller's JWT while serving organization-scoped work (listing or creating access keys, creating users, listing or creating projects), so in practice it surfaces only when the caller's organization has been removed.
