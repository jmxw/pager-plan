# 60012 TemplateSameProjectRequired

> Category: Template | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

A move targeted a template in a different project.

## Error Message

`The project of template({}) and target template({}) must be same.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The source template id. |
| `{}` (2nd) | The target template id. |

## When Returned

Thrown by [`TemplateService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/TemplateService.kt) during `PATCH /v1/template/{templateId}/move_to/{targetTemplateId}` when the source and target [template](/doc/tech_noun/TN0401_template.md) belong to different projects. (The `copy_to` endpoint is a stub and does not raise this yet.)
