# 60007 TemplateDirectoryRequired

> Category: Template | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

An operation required a directory template but the target is a file.

## Error Message

`Template({}) is not a directory.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The template id that was expected to be a directory. |

## When Returned

Thrown by [`TemplateService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/TemplateService.kt) when a directory is required but the target [template](/doc/tech_noun/TN0401_template.md) is a file — reading a directory (`GET /v1/template/{templateId}/directory`) and moving a template into a non-directory target (`PATCH /v1/template/{templateId}/move_to/{targetTemplateId}`).
