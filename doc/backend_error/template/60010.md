# 60010 TemplateBinaryMetadataMissed

> Category: Template | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

A presigned URL could not be produced because the binary object is missing from storage.

## Error Message

`Metadata file for binary template({}) is missed.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The template id. |

## When Returned

Thrown by [`TemplateService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/TemplateService.kt) during `GET /v1/template/{templateId}/binary` when a presigned URL cannot be generated because the binary object is missing from OSS metadata storage.
