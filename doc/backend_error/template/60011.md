# 60011 TemplateRenameReject

> Category: Template | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

Renaming a ROOT or RECYCLE_BIN template is not allowed.

## Error Message

`Cannot rename ROOT or RECYCLE_BIN template({})`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The template id. |

## When Returned

Thrown by [`TemplateService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/TemplateService.kt) during `PATCH /v1/template/{templateId}/rename` when the target [template](/doc/tech_noun/TN0401_template.md) is a `ROOT` or `RECYCLE_BIN` node.
