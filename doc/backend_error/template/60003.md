# 60003 TemplateNotBelongToProject

> Category: Template | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The chosen rendering template belongs to a different project than the article list.

## Error Message

`Template({}) does not belong to project({}).`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The template id. |
| `{}` (2nd) | The project id. |

## When Returned

Thrown by [`ArticleListService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/ArticleListService.kt) during `POST /v1/article_list` when the [template](/doc/tech_noun/TN0401_template.md) chosen to render the [article list](/doc/tech_noun/TN0502_article_list.md) belongs to a different [project](/doc/tech_noun/TN0301_project.md).
