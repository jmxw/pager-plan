# 60008 TemplateUploadFileSaveError

> Category: Template | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

An uploaded template file could not be written to the local cache.

## Error Message

`Error to save uploaded file for template({}).`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The parent template id the file was uploaded under. |

## When Returned

Thrown by [`TemplateService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/TemplateService.kt) during `POST /v1/template/{templateId}/upload` when the uploaded file cannot be saved to the local cache.
