# 60001 TemplateIdNotExist

> Category: Template | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

No template exists for the looked-up id.

## Error Message

`Template id {} not exist.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The template id that was looked up. |

## When Returned

Thrown by [`TemplateRepository`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/repository/TemplateRepository.kt) when a [template](/doc/tech_noun/TN0401_template.md) is resolved by id and no matching row exists. Reached by any endpoint that takes a `{templateId}` or `{targetTemplateId}` path parameter, and when a template is bound to a [navigation](/doc/tech_noun/TN0601_navigation.md) or an [article list](/doc/tech_noun/TN0502_article_list.md).
