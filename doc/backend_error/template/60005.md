# 60005 TemplateNotBinaryFile

> Category: Template | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

A binary-URL request targeted a template that is not a binary file.

## Error Message

`Template({}) is not a binary file.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The template id. |

## When Returned

Thrown by [`TemplateService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/TemplateService.kt) during `GET /v1/template/{templateId}/binary` when the [template](/doc/tech_noun/TN0401_template.md) is not a binary file.
