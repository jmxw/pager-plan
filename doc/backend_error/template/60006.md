# 60006 TemplateNameDuplicated

> Category: Template | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

A template name already exists under the parent directory.

## Error Message

`Template name {} is duplicated in the directory template({}).`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The template name. |
| `{}` (2nd) | The parent directory template id. |

## When Returned

Thrown by [`TemplateService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/TemplateService.kt) when adding (`POST /v1/template/{templateId}/add`) or uploading (`POST /v1/template/{templateId}/upload`) a [template](/doc/tech_noun/TN0401_template.md) whose name already exists under the parent directory.
