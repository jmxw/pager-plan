# 60009 TemplateFileNotExist

> Category: Template | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

A template has no stored file content.

## Error Message

`Template file not exist for template({}).`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The template id. |

## When Returned

Thrown by [`TemplateService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/TemplateService.kt) during `GET /v1/template/{templateId}/binary` when the [template](/doc/tech_noun/TN0401_template.md) has no stored [template file](/doc/tech_noun/TN0402_template_file.md).
