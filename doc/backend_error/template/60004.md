# 60004 TemplateNotTextFile

> Category: Template | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

A text-content operation targeted a template that is not a text file.

## Error Message

`Template({}) is not a text file.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The template id. |

## When Returned

Thrown by [`TemplateService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/TemplateService.kt) when a text-content operation targets a non-text [template](/doc/tech_noun/TN0401_template.md) — reading (`GET /v1/template/{templateId}/content`) or writing (`PATCH /v1/template/{templateId}/content`) the [template file](/doc/tech_noun/TN0402_template_file.md).
