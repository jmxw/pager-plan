# 40007 ProjectZipSavedError

> Category: Project | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The uploaded project archive could not be written to the local cache.

## Error Message

`Error to save project zip file for project({}).`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The project id. |

## When Returned

Thrown by [`ProjectService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/ProjectService.kt) during `POST /v1/project/{projectId}/upload_zip` when the uploaded ZIP cannot be saved to the local cache.
