# 40011 ProjectResetReject

> Category: Project | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

Reset was rejected because the project is being processed.

## Error Message

`Project({}) is processing, try to reset later.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The project id. |

## When Returned

Thrown by [`ProjectService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/ProjectService.kt) during `PATCH /v1/project/{projectId}/reset` when the [project](/doc/tech_noun/TN0301_project.md) state is `ZIP_UPLOADING` or `ZIP_ANALYSING`.
