# 40010 ProjectZipUploadReject

> Category: Project | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

A second archive upload was rejected because the project already has one.

## Error Message

`Reject to upload zip file for project({}), please reset templates!`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The project id. |

## When Returned

Thrown by [`ProjectService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/ProjectService.kt) during `POST /v1/project/{projectId}/upload_zip` when the [project](/doc/tech_noun/TN0301_project.md) is not in `INIT` state (an archive was already uploaded); the project must be reset first.
