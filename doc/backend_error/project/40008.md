# 40008 ProjectZipDownloadFailed

> Category: Project | HTTP Status: N/A — raised only inside the `app/consumer` deployment worker (RabbitMQ context, not an HTTP request). It is logged and produces no HTTP response.

The uploaded project archive could not be obtained for analysis.

## Error Message

`Error to download project zip for project({}) from {}.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The project id. |
| `{}` (2nd) | The OSS pathname the archive was expected at. |

## When Returned

Thrown by [`ProjectZipHandlerService`](/source/pager-backend/app/consumer/src/main/kotlin/com/xwkj/pager/consumer/service/ProjectZipHandlerService.kt) in the consumer when a writable cache file for the [project](/doc/tech_noun/TN0301_project.md) ZIP cannot be prepared for download.
