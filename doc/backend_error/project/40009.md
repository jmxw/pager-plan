# 40009 ProjectZipFileUnzipFailed

> Category: Project | HTTP Status: N/A — raised only inside the `app/consumer` deployment worker (RabbitMQ context, not an HTTP request). It is logged and produces no HTTP response.

The downloaded project archive could not be unzipped.

## Error Message

`Error to unzip file for project({})`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The project id. |

## When Returned

Thrown by [`ProjectZipHandlerService`](/source/pager-backend/app/consumer/src/main/kotlin/com/xwkj/pager/consumer/service/ProjectZipHandlerService.kt) in the consumer when the [project](/doc/tech_noun/TN0301_project.md) ZIP cannot be unzipped or contains no template root directory.
