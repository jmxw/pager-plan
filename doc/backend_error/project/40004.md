# 40004 ProjectIdentifierExist

> Category: Project | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The project identifier chosen at creation is already taken.

## Error Message

`Project identifier {} exist.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The project identifier submitted. |

## When Returned

Thrown by [`ProjectService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/ProjectService.kt) during `POST /v1/project` when the requested [project](/doc/tech_noun/TN0301_project.md) [identifier](/doc/tech_noun/TN0101_identifier.md) already exists.
