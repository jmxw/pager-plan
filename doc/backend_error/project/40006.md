# 40006 ProjectBucketExist

> Category: Project | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

OSS bucket creation for the project identifier failed because a bucket already exists.

## Error Message

`Project bucket exist with identifier {}.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The project identifier (used to derive the bucket names). |

## When Returned

Thrown by [`ProjectService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/ProjectService.kt) during `POST /v1/project` when creating the OSS [bucket](/doc/tech_noun/TN0702_bucket.md) for the [identifier](/doc/tech_noun/TN0101_identifier.md) fails because a bucket with that name already exists.
