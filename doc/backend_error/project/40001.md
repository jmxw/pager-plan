# 40001 ProjectIdNotExist

> Category: Project | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

No project exists for the looked-up id.

## Error Message

`Project id {} not exist.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The project id that was looked up. |

## When Returned

Thrown by [`ProjectRepository`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/repository/ProjectRepository.kt) when a [project](/doc/tech_noun/TN0301_project.md) is resolved by id and no matching row exists. Reached by any endpoint that takes a `{projectId}` path parameter (get / edit / delete project, languages, upload zip, reset, deploy) and by the consumer when handling a deploy or zip message.
