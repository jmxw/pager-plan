# 40002 ProjectNoPrivilege

> Category: Project | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

A developer/customer caller has no privilege granting access to the project.

## Error Message

`No privilege to access project({})`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The project id. |

## When Returned

Thrown by [`PrivilegeComponent`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/component/PrivilegeComponent.kt) when a `DEVELOPER` or `CUSTOMER` caller (OWNER and ADMIN bypass the check) holds no [privilege](/doc/tech_noun/TN0203_privilege.md) for the [project](/doc/tech_noun/TN0301_project.md). Applies to project-scoped endpoints, which authorize through `getByIdWithAuth`.
