# 40012 ProjectIsDeploying

> Category: Project | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

A deploy was requested while the project is already deploying.

## Error Message

`Project({}) is deploying, try to deploy later.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The project id. |

## When Returned

Thrown by [`DeployService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/DeployService.kt) during `PATCH /v1/project/{projectId}/deploy/{phase}` when the [project](/doc/tech_noun/TN0301_project.md) state is `DEPLOYING`. See [deploy task](/doc/tech_noun/TN0701_deploy_task.md).
