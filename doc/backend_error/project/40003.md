# 40003 ProjectNotBelongToOrganization

> Category: Project | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The project exists but belongs to a different organization than the caller.

## Error Message

`Project({}) does not belong to organization({})`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The project id. |
| `{}` (2nd) | The caller's organization id. |

## When Returned

Thrown by [`ProjectRepository`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/repository/ProjectRepository.kt) (`getByIdWithAuth`) when the [project](/doc/tech_noun/TN0301_project.md) resolves but its [organization](/doc/tech_noun/TN0201_organization.md) is not the caller's. Applies to project-scoped endpoints.
