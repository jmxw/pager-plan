# 90008 NavigationRootDeleteReject

> Category: Navigation | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The root navigation cannot be deleted.

## Error Message

`Root navigation({}) cannot be deleted.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The navigation id. |

## When Returned

Thrown by [`NavigationService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/NavigationService.kt) during `DELETE /v1/navigation/{navigationId}` when the target is the project's `ROOT` [navigation](/doc/tech_noun/TN0601_navigation.md).
