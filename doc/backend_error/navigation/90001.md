# 90001 NavigationIdNotExist

> Category: Navigation | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

No navigation exists for the looked-up id.

## Error Message

`Navigation id {} not exist`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The navigation id that was looked up. |

## When Returned

Thrown by [`NavigationRepository`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/repository/NavigationRepository.kt) when a [navigation](/doc/tech_noun/TN0601_navigation.md) is resolved by id and no matching row exists. Reached by any endpoint that takes a `{navigationId}` or `{targetNavigationId}` path parameter (get, children, add list / article, move, up, down, delete).
