# 90005 NavigationTemplateMismatching

> Category: Navigation | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The chosen rendering template belongs to a different project than the navigation.

## Error Message

`Navigation({}} and template({}) must belong to a same project`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The navigation id. (The doubled `}` in `Navigation({}}` is an upstream typo.) |
| `{}` (2nd) | The rendering template. The `Template` entity is passed, so its default string form is substituted (not the bare id). |

## When Returned

Thrown by [`NavigationListService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/NavigationListService.kt) and [`NavigationArticleService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/NavigationArticleService.kt) during `POST /v1/navigation/{navigationId}/list` and `POST /v1/navigation/{navigationId}/article` when the chosen [template](/doc/tech_noun/TN0401_template.md) belongs to a different [project](/doc/tech_noun/TN0301_project.md) than the [navigation](/doc/tech_noun/TN0601_navigation.md).
