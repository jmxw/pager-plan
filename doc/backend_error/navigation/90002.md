# 90002 NavigationIdentifierDuplicated

> Category: Navigation | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

A sibling navigation under the parent already uses the identifier.

## Error Message

`Navigation identifier {} is duplicated in the parent navigation({}).`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The navigation identifier submitted. |
| `{}` (2nd) | The parent navigation id. |

## When Returned

Thrown by [`NavigationListService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/NavigationListService.kt) and [`NavigationArticleService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/NavigationArticleService.kt) during `POST /v1/navigation/{navigationId}/list` and `POST /v1/navigation/{navigationId}/article` when a sibling [navigation](/doc/tech_noun/TN0601_navigation.md) under the parent already uses the [identifier](/doc/tech_noun/TN0101_identifier.md).
