# 90007 NavigationDeleteRejectWithChildren

> Category: Navigation | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

A navigation with child navigations cannot be deleted.

## Error Message

`Navigation({}) with children navigations cannot be deleted.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The navigation id. |

## When Returned

Thrown by [`NavigationService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/NavigationService.kt) during `DELETE /v1/navigation/{navigationId}` when the [navigation](/doc/tech_noun/TN0601_navigation.md) still has child navigations.
