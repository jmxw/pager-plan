# 90004 NavigationArticleListMismatching

> Category: Navigation | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The chosen article list belongs to a different project than the navigation.

## Error Message

`Navigation({}} and article list({}) must belong to a same project`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The navigation id. (The doubled `}` in `Navigation({}}` is an upstream typo.) |
| `{}` (2nd) | The article list id. |

## When Returned

Thrown by [`NavigationListService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/NavigationListService.kt) during `POST /v1/navigation/{navigationId}/list` when the chosen [article list](/doc/tech_noun/TN0502_article_list.md) belongs to a different [project](/doc/tech_noun/TN0301_project.md) than the [navigation](/doc/tech_noun/TN0601_navigation.md).
