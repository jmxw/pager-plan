# 90003 NavigationArticleMismatching

> Category: Navigation | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The chosen article belongs to a different project than the navigation.

## Error Message

`Navigation({}} and article({}) must belong to a same project`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The navigation id. (The message text reads `Navigation({}}` — the doubled `}` is an upstream typo; the first `{}` is the id and the extra `}` is literal.) |
| `{}` (2nd) | The article id. |

## When Returned

Thrown by [`NavigationArticleService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/NavigationArticleService.kt) during `POST /v1/navigation/{navigationId}/article` when the chosen [article](/doc/tech_noun/TN0501_article.md) belongs to a different [project](/doc/tech_noun/TN0301_project.md) than the [navigation](/doc/tech_noun/TN0601_navigation.md).
