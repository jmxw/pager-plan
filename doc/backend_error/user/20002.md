# 20002 UserNotPermitted

> Category: User | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The caller's role does not hold the permission required by the endpoint.

## Error Message

`User({}) with role {} does not have permission {}.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The caller's user id. |
| `{}` (2nd) | The caller's `UserRole`. |
| `{}` (3rd) | The `UserPermission` required by the endpoint. |

## When Returned

Thrown by [`UserPermissionInterceptor`](/source/pager-backend/app/cms/src/main/kotlin/com/xwkj/cms/security/UserPermissionInterceptor.kt) before the handler runs when the caller's role does not contain the endpoint's `@Permitted(UserPermission.X)` value. Applies to every permission-gated endpoint; see [permission](/doc/permission/README.md) for the role-to-permission mapping.
