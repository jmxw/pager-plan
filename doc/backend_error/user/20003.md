# 20003 UserIdentifierExist

> Category: User | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The user identifier chosen when creating an account is already taken.

## Error Message

`User identifier {} exist.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The user identifier submitted. |

## When Returned

Thrown by [`UserService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/UserService.kt) when creating a [user](/doc/tech_noun/TN0202_user.md) (`POST /v1/user/admin`, `POST /v1/user/developer`, `POST /v1/user/customer`) and by [`OrganizationService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/OrganizationService.kt) when creating the owner account during `POST /v1/organization/register`, when the [identifier](/doc/tech_noun/TN0101_identifier.md) already exists.
