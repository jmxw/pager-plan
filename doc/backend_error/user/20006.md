# 20006 UserMailExist

> Category: User | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The mail address chosen when creating an account is already used in the organization.

## Error Message

`User mail {} exist in organization {}.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The submitted **user identifier** — despite the wording "mail", [`UserService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/UserService.kt) passes the identifier here. |
| `{}` (2nd) | The organization. **Not substituted:** no second argument is supplied, so this placeholder renders literally as `{}`. |

## When Returned

Thrown by [`UserService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/UserService.kt) when creating a [user](/doc/tech_noun/TN0202_user.md) (`POST /v1/user/admin`, `POST /v1/user/developer`, `POST /v1/user/customer`) and the submitted mail already exists. Note the message/argument mismatch documented under Placeholders.
