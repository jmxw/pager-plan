# 20005 UserPasswordWrong

> Category: User | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

Login was attempted with a wrong password for an existing user.

## Error Message

`Password is wrong for user {}`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The login identifier submitted. |

## When Returned

Thrown by [`UserService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/UserService.kt) during `POST /v1/user/login` when the [identifier](/doc/tech_noun/TN0101_identifier.md) resolves to a [user](/doc/tech_noun/TN0202_user.md) but the submitted password does not match the stored credential.
