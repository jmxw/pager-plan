# 20001 UserIdNotExist

> Category: User | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

No user exists for the looked-up id.

## Error Message

`User id {} not exist.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The user id that was looked up. |

## When Returned

Thrown when the caller's own account is resolved from the JWT and no matching [user](/doc/tech_noun/TN0202_user.md) exists — by [`UserRepository`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/repository/UserRepository.kt) (e.g. `GET /v1/user/me`, article authoring, deploy) and by [`PrivilegeComponent`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/component/PrivilegeComponent.kt) when collecting a non-admin caller's projects. In practice it surfaces only when the token's user has been deleted.
