# 20004 UserIdentifierNotExist

> Category: User | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

Login was attempted with an identifier that matches no user.

## Error Message

`User identifier {} not exist.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The login identifier submitted. |

## When Returned

Thrown by [`UserService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/UserService.kt) during `POST /v1/user/login` when no [user](/doc/tech_noun/TN0202_user.md) has the submitted [identifier](/doc/tech_noun/TN0101_identifier.md).
