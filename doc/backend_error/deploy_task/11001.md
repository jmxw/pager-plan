# 11001 DeployTaskIdNotExist

> Category: Deploy Task | HTTP Status: N/A — raised only inside the `app/consumer` deployment worker (RabbitMQ context, not an HTTP request). It is logged and produces no HTTP response.

No deploy task exists for the id carried by a deployment message.

## Error Message

`Deploy task id {} not exist.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The deploy task id. |

## When Returned

Thrown by [`DeployTaskRepository`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/repository/DeployTaskRepository.kt) when a [deploy task](/doc/tech_noun/TN0701_deploy_task.md) is resolved by id and no matching row exists. Reached only from the consumer while processing a deployment message.
