# 11004 DeployDirectoryError

> Category: Deploy Task | HTTP Status: N/A — raised only inside the `app/consumer` deployment worker (RabbitMQ context, not an HTTP request). It is logged and produces no HTTP response.

The local working directory for the deploy could not be created.

## Error Message

`Error to find or create a deploy directory for project({})`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The project id. |

## When Returned

Thrown by [`ProjectDeployService`](/source/pager-backend/app/consumer/src/main/kotlin/com/xwkj/pager/consumer/service/ProjectDeployService.kt) in the consumer when the local working directory for the [deploy task](/doc/tech_noun/TN0701_deploy_task.md) of the [project](/doc/tech_noun/TN0301_project.md) cannot be found or created.
