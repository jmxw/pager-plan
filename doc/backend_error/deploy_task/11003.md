# 11003 DeployTaskSkipDueToNoTemplate

> Category: Deploy Task | HTTP Status: N/A — raised only inside the `app/consumer` deployment worker (RabbitMQ context, not an HTTP request). It is logged and produces no HTTP response.

A deploy task was skipped because the project's template is not ready.

## Error Message

`Deploy task({}) skipped, because the template of the project({}) is not ready.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The deploy task. The `DeployTask` entity is passed, so its default string form is substituted (not the bare id). |
| `{}` (2nd) | The project id. |

## When Returned

Thrown by [`ProjectDeployService`](/source/pager-backend/app/consumer/src/main/kotlin/com/xwkj/pager/consumer/service/ProjectDeployService.kt) in the consumer when the [project](/doc/tech_noun/TN0301_project.md) has no ready root [template](/doc/tech_noun/TN0401_template.md) (its state is earlier than `TEMPLATE_READY`).
