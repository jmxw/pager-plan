# 70010 ArticleDeleteRejectedDueToPagerListExisting

> Category: Article List | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

Deletion was rejected because the article list is still referenced by templates.

## Error Message

`Article list({}) cannot be deleted, because it was referenced by more that 1 templates.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The article list being deleted. The `ArticleList` entity is passed, so its default string form is substituted (not the bare id). |

## When Returned

Thrown by [`ArticleListService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/ArticleListService.kt) during `DELETE /v1/article_list/{articleListId}` when the [article list](/doc/tech_noun/TN0502_article_list.md) is still referenced by precompiled [pager tag](/doc/tech_noun/TN0403_pager_tag.md) occurrences in templates.
