# 70005 ArticleListTemplateFormatError

> Category: Article List | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The chosen rendering template is not an HTML file.

## Error Message

`Article template must be html.`

## Placeholders

N/A — the message contains no placeholder.

## When Returned

Thrown by [`ArticleListService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/ArticleListService.kt) during `POST /v1/article_list` when the [template](/doc/tech_noun/TN0401_template.md) chosen to render the [article list](/doc/tech_noun/TN0502_article_list.md) is not an HTML/HTM file.
