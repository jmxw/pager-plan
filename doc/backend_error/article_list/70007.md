# 70007 ArticleListArticleMismatching

> Category: Article List | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The article and article list belong to different projects.

## Error Message

`Article list({}) and article({}) is mismatching, they must belong to a same project.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The article list id. |
| `{}` (2nd) | The article id. |

## When Returned

Thrown by [`ArticleListService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/ArticleListService.kt) during `PATCH` / `DELETE /v1/article_list/{articleListId}/article/{articleId}` when the [article](/doc/tech_noun/TN0501_article.md) and the [article list](/doc/tech_noun/TN0502_article_list.md) belong to different projects.
