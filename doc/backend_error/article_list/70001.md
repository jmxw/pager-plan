# 70001 ArticleListIdNotExist

> Category: Article List | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

No article list exists for the looked-up id.

## Error Message

`Article list id {} not exist.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The article list id that was looked up. |

## When Returned

Thrown by [`ArticleListRepository`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/repository/ArticleListRepository.kt) when an [article list](/doc/tech_noun/TN0502_article_list.md) is resolved by id and no matching row exists. Reached by any endpoint that takes an `{articleListId}` path parameter and when an article list is bound to a list [navigation](/doc/tech_noun/TN0601_navigation.md).
