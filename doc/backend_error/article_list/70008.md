# 70008 ArticleListArticleExist

> Category: Article List | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The article is already contained in the article list.

## Error Message

`Article list({}) contains article({}).`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The article list id. |
| `{}` (2nd) | The article id. |

## When Returned

Thrown by [`ArticleListService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/ArticleListService.kt) during `PATCH /v1/article_list/{articleListId}/article/{articleId}` when the [article](/doc/tech_noun/TN0501_article.md) is already in the [article list](/doc/tech_noun/TN0502_article_list.md).
