# 70006 ArticleListProjectMismatching

> Category: Article List | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

One or more requested article lists do not belong to the given project.

## Error Message

`Article list and project is mismatching.`

## Placeholders

N/A — the message contains no placeholder.

## When Returned

Thrown by [`ArticleListRepository`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/repository/ArticleListRepository.kt) (`findByIdsInProjectWithAuth`) when one or more of the requested [article list](/doc/tech_noun/TN0502_article_list.md) do not belong to the given [project](/doc/tech_noun/TN0301_project.md).
