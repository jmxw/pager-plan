# 70003 ArticleListIdentifierExist

> Category: Article List | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The article list identifier chosen at creation already exists in the project.

## Error Message

`Article list identifier {} exist in project({}).`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The article list identifier submitted. |
| `{}` (2nd) | The project id. |

## When Returned

Thrown by [`ArticleListService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/ArticleListService.kt) during `POST /v1/article_list` when the [article list](/doc/tech_noun/TN0502_article_list.md) [identifier](/doc/tech_noun/TN0101_identifier.md) already exists in the [project](/doc/tech_noun/TN0301_project.md).
