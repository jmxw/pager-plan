# 50002 LanguageNotExist

> Category: Language | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

A specific language was requested for a project that does not have it enabled.

## Error Message

`Language {} not exist in project({}).`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The requested `LanguageType`. |
| `{}` (2nd) | The project id. |

## When Returned

Thrown by [`LanguageRepository`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/repository/LanguageRepository.kt) when a specific (non-default) [language](/doc/tech_noun/TN0302_language.md) is requested for a [project](/doc/tech_noun/TN0301_project.md) that has not enabled it.
