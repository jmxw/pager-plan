# 50001 LanguageExist

> Category: Language | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

The language being added is already enabled for the project.

## Error Message

`Language {} exist in project({}).`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` (1st) | The `LanguageType` being added. |
| `{}` (2nd) | The project id. |

## When Returned

Thrown by [`LanguageService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/LanguageService.kt) during `POST /v1/project/{projectId}/language` when the [language](/doc/tech_noun/TN0302_language.md) is already enabled for the [project](/doc/tech_noun/TN0301_project.md).
