# 80003 ArticleCoverSaveError

> Category: Article | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

An uploaded article cover could not be written to the local cache.

## Error Message

`Error to save cover file for article({}).`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The article id. |

## When Returned

Thrown by [`ArticleService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/ArticleService.kt) during `POST /v1/article/{articleId}/cover` when the uploaded cover cannot be saved to the local cache.
