# 80001 ArticleIdNotExist

> Category: Article | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

No article exists for the looked-up id.

## Error Message

`Article id {} not exist.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The article id that was looked up. |

## When Returned

Thrown by [`ArticleRepository`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/repository/ArticleRepository.kt) when an [article](/doc/tech_noun/TN0501_article.md) is resolved by id and no matching row exists. Reached by any endpoint that takes an `{articleId}` path parameter and when an article is bound to an [article list](/doc/tech_noun/TN0502_article_list.md) or an article [navigation](/doc/tech_noun/TN0601_navigation.md).
