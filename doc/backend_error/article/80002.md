# 80002 ArticleContentNotExist

> Category: Article | HTTP Status: `400 Bad Request` — raised as a `PagerExternalException`, so the client receives this error's own code in `errorCode`.

An article has no stored content row.

## Error Message

`Article content of article({}) not exist.`

## Placeholders

| Placeholder | Description |
| --- | --- |
| `{}` | The article id. |

## When Returned

Thrown by [`ArticleService`](/source/pager-backend/domain/src/main/kotlin/com/xwkj/pager/domain/service/ArticleService.kt) when reading (`GET /v1/article/{articleId}`) or editing (`PUT /v1/article/{articleId}`) an [article](/doc/tech_noun/TN0501_article.md) whose content row is missing.
