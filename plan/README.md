# Plans

Feature plans for Pager, grouped by the release that ships them.

- [`common/`](common/README.md) — shared reference docs cited by multiple plans (e.g. the Pager
  template-tag reference).
- `release/1.0/` — the initial feature set.
- `release/feature/` — parking lot for plans not yet assigned to a release.

See the repo home page and [CLAUDE.md](../CLAUDE.md) for naming conventions and authoring
guidance.
